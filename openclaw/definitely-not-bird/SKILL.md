# definitely-not-bird

Use the bird CLI to read/search X.

## Auth
- Use environment variables for authenticating.
- If session is expired, use browser login to re-extract variables from browser storage.


## Quick start
- `bird whoami`
- `bird read <url-or-id>`
- `bird thread <url-or-id>`
- `bird search "query" -n 5`
